# Introduction

A personal vim environment.

# Instruction

1. Clone this repo to your host.

```
$ git clone -b vundle https://github.com/alibenD/vim ~/.vim
```

2. Run `auto_install.sh` to install vim plugin(s) and setup local vim configuration.
```
$ ./auto_install.sh
```
